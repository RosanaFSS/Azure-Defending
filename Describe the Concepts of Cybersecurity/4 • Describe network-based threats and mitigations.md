<p>In a network, what is the fundamental building block that allows multiple devices to communicate with each other?<br>
<code>The switch</code></p>

<p>What is a datagram used for?<br>
<code>A datagram is a chunk of data of uniform length that can be sent across the network.</code></p>

<p>What is the name of a common network attack where the cybercriminal compromises a router in the network to eavesdrop on, or alter, data?<br>
<code>Man-in-the-middle.</code></p>

<p>When considering security, what is the common name given to separating your network into discrete entities?<br>
<code>Network segmentation.</code></p>
